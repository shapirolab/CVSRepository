Welcome to the BioSpi Download Page

To download the most recent release of the system, right-click aspic-release.tar.gz and select Save Link (Target) as ... in Netscape or Save Target as ... in Microsoft Internet Explorer. Then browse to the directory to which it is to be copied, and select Save (or Save as).

To expand the release, enter:

    % zcat aspic-release.tar.gz | tar -xf -
A directory named Aspic-release and a system of sub-directories will be expanded in the current directory.

A release includes emulators for linux, sgi and solaris platforms.
To install it on one or more of those processors, see Aspic-release/README.

Additional documentation for the Logix system is The Logix System User Manual Version 2.0, Supplement to User Manual for System 2.2 and Course.tar.gz, material from a course given in 1986 and 1987.
